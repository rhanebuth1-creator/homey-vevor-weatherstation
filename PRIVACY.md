# Datenschutzerklärung / Privacy Policy

## Deutsch

Diese App kommuniziert ausschließlich mit der Tuya Cloud API (openapi.tuyaeu.com) um Sensordaten deiner Wetterstation abzurufen. Es werden keine persönlichen Daten gespeichert oder an Dritte weitergegeben.

Die App benötigt folgende Zugangsdaten die lokal auf deinem Homey Pro gespeichert werden:
- Tuya Access ID und Access Secret (aus deinem Tuya Developer Account)
- Tuya Device ID deiner Wetterstation

Diese Daten verlassen deinen Homey Pro nur um die Verbindung zur Tuya Cloud API herzustellen.

## English

This app communicates exclusively with the Tuya Cloud API (openapi.tuyaeu.com) to retrieve sensor data from your weather station. No personal data is stored or shared with third parties.

The app requires the following credentials which are stored locally on your Homey Pro:
- Tuya Access ID and Access Secret (from your Tuya Developer Account)
- Tuya Device ID of your weather station

This data only leaves your Homey Pro to establish the connection to the Tuya Cloud API.
