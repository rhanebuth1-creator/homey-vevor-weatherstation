Vevor YT60307 Weather Station for Homey

Integrate your Vevor YT60307 7-in-1 Wi-Fi weather station into Homey via the Tuya Cloud API.

FEATURES
- Indoor & outdoor temperature
- Indoor & outdoor humidity
- Wind speed & wind direction
- Rainfall
- Barometric pressure
- UV index
- Light intensity
- Pool/channel temperature

REQUIREMENTS
- Vevor YT60307 weather station
- Free Tuya Developer account (iot.tuya.com)
- Tuya Access ID, Access Secret and Device ID

SETUP
1. Create a free account at iot.tuya.com
2. Create a Cloud project and link your Smart Life app account
3. Note your Access ID, Access Secret and Device ID
4. Add the device in Homey and enter the credentials in settings
